Introduction JavaScript

JavaScript― это интрепретируемый общий языка программирования общего назанчения с динамической тпипзацией данных.

Интерпретирумый значит программа выполянется в исходном виде, т.е., исходный код не преобразуется в машинный код или промежуточный код. В связи с этим ошибок на этапе компиляции и компоновки отсутсвует, есть только ошибки на этапе выполнения. FrontEnd

Динамическая типизация говорит о том, что тип переменных и констант определяется во время присваивания, по типу значения справа (r-value). l-value = r-value; l-value - выражение слева; r-value - выражение справа;

Перменные и типы данных

var
объявляет глоабльную переменную, несмотря на область видимости, в которой она объявлена. Если записать значение в несуществующую перменную, то это не вызовет ошибку, это будет аналогично ключевому слову 'var';
let
объявляет локальную переменную, только для той области видимости, в которой она используется;
const
объявляет константу. Константы могут быть только локальными;
