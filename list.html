Списки

Нумерованный список

Список определений

Позволяет создавать списки терминов с их разъяснением

<dl> ― Definitions List;
<dt> ― Description Term;
<dd> ― Description Details;
<dt> и <dd> всегда используются парой.
Нумерованый список
Позволяет нумеровать элементы тектса. Это нужно для перечисления в строгом порядке.
Марикрованный список
Позволяет перечислить элементы текста в любом порядке.
Список определений
Позволяет перечислять понятия и давать им определения.

Маркированный список

<ol> ― Unordered List;
<li> ― List Item;
  1. List Item 1
  2. List Item 2
  3. List Item 3
  4. List Item 4
  5. List Item 5
